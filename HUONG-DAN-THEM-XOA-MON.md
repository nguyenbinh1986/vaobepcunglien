# 📝 Hướng dẫn THÊM / XÓA / SỬA món (cho vợ)

Trang quản trị: **https://vaobepcunglien.vercel.app/admin.html**

> ✨ **Quy trình MỚI cực đơn giản:** mọi thay đổi trong admin **tự lưu vào web NGAY**.
> Không cần tải file, không cần GitHub, không cần chạy SQL gì nữa!

---

## ➕ THÊM MÓN
1. Mở admin → đăng nhập → bấm **+ Thêm món**
2. Điền: tên, nhóm, **giá**, ảnh (nếu có), giới thiệu, **link video**, nguyên liệu, các bước, mẹo
3. Bấm **🎲 Tạo mã** (hoặc tự đặt mã) → bấm **Lưu món**
4. **Xong!** Mở web lên xem là thấy món mới ngay.

## ✏️ SỬA MÓN (đổi giá, tên, công thức, video…)
1. Bấm **✏️ Sửa** ở món → đổi gì tuỳ ý → **Lưu món**
2. **Xong!** Web đổi ngay.

## 🗑️ XÓA MÓN
1. Bấm **🗑** ở món → xác nhận
2. **Xong!** Món biến mất khỏi web ngay.

## ⚙️ ĐỔI THÔNG TIN TRANG / TÀI KHOẢN
- Sửa ở khung "Thông tin trang & thanh toán" → **tự lưu** (hoặc bấm **💾 Lưu cài đặt**).

---

## 💡 Ghi nhớ
- **Mã mở khóa:** khách trả tiền là web **tự mở**, không cần mã. Mã chỉ dùng để gửi tay khi cần hỗ trợ — bấm **🔑 Tải bảng mã** để lưu lại danh sách mã (giữ riêng).
- **Đăng xuất** khi dùng máy lạ.
- Video nên để YouTube **"Không công khai"** cho an toàn.

> Bí chỗ nào cứ nhắn Bình (người dựng web).
